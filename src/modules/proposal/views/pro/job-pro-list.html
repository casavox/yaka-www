VOUS N’AVEZ AUCUN CHANTIER EN COURS Comment obtenir de nouveaux chantiers ?

Faire une offre
  • Vérifiez la validité de votre profil en cliquant sur votre logo avatar depuis la barre de menu Mon profil Pro
  • Sélectionnez un projet depuis la Carte des offres
    • Naviguer et zoomer dans cette carte pour retrouver les offres correspondant à votre profil
    • Sélectionnez un projet pour en voir les détails
    • Faites votre proposition : Si vous avez assez d'élément, indiquez un montant approximatif et une date d'intervention, puis ajouter un premier message afin de vous mettre en avant et d’engager la discussion avec votre client.
  • Losrque vous remportez une offre, retrouvez-la ici, dans « Mes chantiers »
Suivre mon chantier
  • Vous pourrez retrouver les coordonnées complètes du client sur la fiche projet (même une fois le projet clot)
  • Vous pouvez à tout moment mettre à jour la date d'intervention et le prix définitif convenu avec le client
  • Vous pourrez biensûr continuer d'utiliser les services CasaVox pour vos échanges privés avec le client pendant toute la durée des travaux (messages, photos, plans, devis, factures, etc.), l'historique de ces échanges reste accessible même une fois le projet clot
  • Dès qu'un chantier est terminé, pensez à relancer votre client pour qu'il clot son projet, cette action augmente votre visibilité sur CasaVox en permettant de vous faire connaitre auprès des connaissances de ce client !
